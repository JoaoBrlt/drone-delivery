# Demonstration script

## Requirements

- Install the dependencies:

```bash
pip install -r requirements.txt
```

## Scenario 1

- Run the first scenario:

```bash
python demo.py -n 1
```

> Use the `Right arrow` key to run the scenario step by step.

## Scenario 2

- Run the second scenario:

```bash
python demo.py -n 2
```

> Use the `Right arrow` key to run the scenario step by step.
